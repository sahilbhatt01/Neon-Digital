## 🧠 What is Neon Digital?

**Neon Digital** is not just a website — it’s a **frontend experience system**.

Built using **only core web technologies**, it focuses on:

* ⚡ Speed without frameworks
* 🎯 Clean architecture
* 🎬 Cinematic UI interactions
* 🌌 Immersive visual depth

> Designed like a product. Built like a system.

## ⚙️ Engine Breakdown

### 🔹 Component Injection

```js
fetch("header.html")
fetch("footer.html")
```

Reusable layout → zero duplication → scalable structure

---

### 🔹 Scroll Animation Engine

```css
.reveal.active
```

Viewport-triggered animations → smooth UX

---

### 🔹 Particle System

* Canvas-based rendering loop
* Lightweight animation engine
* Creates depth + motion
